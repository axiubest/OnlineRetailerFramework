# OnlineRetailerFramework
电子商务框架
简单封装电商类常用控件，首页瀑布流信息展示，详情页，加载更多页，购物车，个人中心，登录等，持续集成更新中， 内有catogry分类等，封装多年积累代码，有空持续更新中，仅供学习交流使用，请勿用于任何商业用途
